Commencer par lancer le fichier "check_packages.py" pour verifier que vous avez installé tous les packages utilisés.
        python check_packages.py

Pour lancer l'application streamlit lancer le code suivant :
        streamlit run Accueil.py

NB : le package "Jax" n'est pas compatible avec les processeurs autre que Intel. Il n'est donc pas compatible avec les Mac M1 
ou plus récents.
